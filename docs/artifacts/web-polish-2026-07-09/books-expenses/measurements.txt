── Constant-height workbar (Task 3) ──────────────────────────────────────────
Pinned toolbar wrapper (the sticky div TableChrome publishes as --shell-header-top),
measured after a full reload + settle per segment:

  invoices    96px   (Row1 filter chips 31px + Row2 tab strip 28px)
  estimates   ~96px  (same structure as invoices)
  expenses    97px   (Row1 month chips 32px + Row2 tab strip 28px)
  sync        97px   (Row1 status badge pinned to 32px + Row2 tab strip 28px)

All four within 1px → the pinned chrome no longer jumps on segment switch.

NOTE: before the sync fix, sync connected = 93px (badge Row1 = 28px) and, during
the connections-query load window, the "NOT CONNECTED" readout made Row1 ~18px
(toolbar ~79px). Fix: sync's meta cell is pinned to min-h-[32px] so Row1 matches
the list-surface chip row in EVERY connection state (connected / offline / not
connected).
