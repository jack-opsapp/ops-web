Approve this layout direction

Persistent list + resizable divider + sticky segmented picker (ALL/EMAIL/PORTAL) below thread header + chat bubbles + channel dividers inline + collapsible context panel (not shown, slides from right). Reply bar defaults to last-used channel with a dropdown to switch.

Key details in this mockup

  • Conversation list — sorted by recency, client avatars with initials, channel badge on preview, unread count, unmatched emails styled distinctly (? avatar, orange badge)
  • Segmented picker — sticky below the thread header, pipeline-style with bottom-border indicator on active tab
  • Chat bubbles — left-aligned (inbound), right-aligned (outbound). Subtle accent tint on outbound. Timestamps below bubbles.
  • Channel dividers — inline separators with icon + label when switching between email threads and portal messages in "ALL" view
  • Reply bar — channel selector dropdown (defaults to last message's channel), text input, attach, send
  • Resize handle — visible grab indicator between panels
